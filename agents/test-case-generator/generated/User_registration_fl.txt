 - Test Case Title: Verify user registration with valid details
  - Steps to Reproduce:
    - User opens the homepage
    - User clicks on the "Sign Up" button
    - User enters a unique email address in the email field
    - User enters a strong password in the password field
    - User confirms the password by typing the same strong password in the confirm password field
    - User clicks the "Sign Up" button
  - Expected Result:
    - An account creation success message is displayed
    - The user is redirected to the homepage

- Test Case Title: Verify user registration with inactive account
  - Steps to Reproduce:

    - User clicks on the "Sign In" button

    - User clicks the "Sign In" button
  - Expected Result:
